常用数据结构与时间复杂度
时间复杂度：O(N)
空间复杂度：O(N)

常见的时间复杂度(时间复杂度从小到大为)：
1.常数阶：O(1)
2.对数阶：O(log2N)
3.线性阶：O(n)
4.线性对数阶：O(nlogN)
5.幂函数阶：O(n^2)/O(n^3)....
6.指数阶：O(2^n)


# 常用数据结构的时间复杂度：
数据结构 查找 插入 删除 遍历

1. 数组 O(N) O(1) O(N) ----
2. 有序数组 O(logN) O(N) O(N) O(N)
3. 链表 O(N) O(1) O(N) ----
4. 有序链表 O(N) O(N) O(N) O(N)
5. 二叉树（一般情况） O(logN) O(logN) O(logN) O(N)
6. 二叉树（最坏情况） O(N) O(N) O(N) O(N)
7. 平衡树（一般和最坏） O(logN) O(logN) O(logN) O(N)
8. 哈希表 O(1) O(1) O(1) ----
9. 二叉搜索树  O(h) O(h) O(h) O(N)
